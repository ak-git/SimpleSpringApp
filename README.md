# Simple SpringBoot App with Docker support

## Build docker image

`docker build -t a002k/simple-spring-app:latest .`

## Run docker image

`docker run --rm a002k/simple-spring-app`

## Run docker image as docker-compose

`docker-compose run --rm simple-spring-app`
